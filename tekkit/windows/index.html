Tekkit install instructions for Windows. Please begin by downloading the file below.

Download TechnicLauncher.exe

Server IP: tekkit.darcraft.net

Installation Instructions

1. Download the TechnicPack.exe file by clicking on the green button above.
2. Drag the executable file to a desireable location on your computer, preferrably your desktop or your documents folder.
3. Launch the executable file by double clicking on it
4. Click open or continue if prompted with any security prompts.
5. Login with your Minecraft username and password. If you have migrated your account to a Mojang account, use your email address.
6. Select the 'Tekkit' modpack with the icon that looks like [this]. Use the up and down arrow keys to do so.
7. Click Play and wait for the files to download. This may take several minutes, but will only happen once.
8. Click Multiplayer then Add Server. Enter the IP 'tekkit.darcraft.net' and select save.
9. Select the newly added server and click Join Server.
